# 이클립스 단축키 가이드

## 태그 선택 단축키

### HTML/XML 태그 쌍 선택

#### Alt+Shift+A
- 여는 태그와 닫는 태그를 동시에 선택
- HTML/XML 에디터에서 사용 가능
- 커서가 태그 내부에 있을 때 동작
- 해당 단축키를 다시 누르면 선택 해제 (Toggle 기능)

#### 더블 클릭
- 태그 이름을 더블 클릭하면 해당 태그 쌍 선택
- `<div>`를 더블 클릭하면 여는 `<div>`와 닫는 `</div>` 모두 선택

## 관련 단축키

### 선택 확장
- **Ctrl+Shift+→**: 다음 단어 선택
- **Ctrl+Shift+←**: 이전 단어 선택
- **Ctrl+Shift+Home**: 문서 시작까지 선택
- **Ctrl+Shift+End**: 문서 끝까지 선택

### 코드 편집
- **Ctrl+D**: 한 줄 삭제
- **Ctrl+L**: 한 줄 선택
- **Alt+Shift+↑**: 현재 줄 복사
- **Alt+Shift+↓**: 현재 줄 이동

### 코드 정렬
- **Ctrl+Shift+F**: 코드 자동 정렬 (Format)
  - 현재 파일 전체 또는 선택된 영역을 자동으로 정렬
  - Java, XML, HTML, JavaScript 등 다양한 파일 타입 지원
  - 코드 스타일 일관성 유지에 필수적인 단축키

### 탐색
- **Ctrl+Shift+O**: Import 정리
- **Ctrl+Space**: 자동 완성
- **F3**: 선언부로 이동

## 참고사항

- Alt+Shift+A는 일부 키보드 레이아웃에서 한글 입력 시 문제가 발생할 수 있음
- 이 경우 Window → Preferences → General → Keys에서 검색 후 다른 단축키로 변경 가능
- "Select Enclosing Element" 명령으로 검색
