# IDE 단축키 공통 패턴

다양한 IDE(이클립스, VSCode 등)에서 발견되는 단축키 공통 패턴과 구조를 분석하고 종합한 문서입니다.

## 핵심 패턴

### 1. 선택 확장 패턴 (Selection Expansion)

가장 보편적인 단축키 패턴으로, 거의 모든 IDE에서 동일하게 사용됩니다:

#### 단어 단위 선택
- **Ctrl+Shift+→**: 다음 단어 선택
- **Ctrl+Shift+←**: 이전 단어 선택

#### 문서 단위 선택
- **Ctrl+Shift+Home**: 문서 시작까지 선택
- **Ctrl+Shift+End**: 문서 끝까지 선택

이 패턴은 모든 주요 IDE에서 **완전히 동일**하게 작동하며, 가장 기본적인 텍스트 조작 단축키입니다.

---

### 2. 제어 키 조합 패턴 (Modifier Key Patterns)

#### Ctrl 기반 패턴
- **Ctrl+Space**: 자동 완성 (거의 모든 IDE 공통)
- **Ctrl+D**: VSCode에서 다음 동일 단어 선택, 이클립스에서 한 줄 삭제
- **Ctrl+/**: 주석 토글

#### Shift 조합 패턴 (확장/반대)
- **Ctrl+Shift+[방향키]**: 선택 영역 확장
- **Shift+Alt+[방향키]**: 줄 이동/복사 (VSCode 중심)

#### Alt 조합 패턴 (대체/옵션)
- **Alt+[방향키]**: 줄 이동 (VSCode), 블록 선택 (일부 IDE)
- **Alt+Shift+[방향키]**: 줄 복사, 태그 선택

---

### 3. 기능 카테고리별 공통점

#### 태그/블록 선택
| 기능 | 이클립스 | VSCode | 공통점 |
|------|---------|--------|--------|
| 더블 클릭 태그 선택 | ✓ | ✓ | 가장 직관적 |
| 태그 쌍 확장 | Alt+Shift+A | Alt+Shift+→ | Alt+Shift 기반 |
| 토글 방식 | ✓ | ✓ | 반복 입력으로 해제 |

#### 코드 정렬
| 기능 | 이클립스 | VSCode | 공통점 |
|------|---------|--------|--------|
| 전체 정렬 | Ctrl+Shift+F | Shift+Alt+F | Shift+[수정자]+F |
| 선택 영역 정렬 | - | Ctrl+K, Ctrl+F | 복합키 사용 |
| 자동 포맷터 연동 | ✓ | ✓ | 확장/플러그인 기반 |

#### 줄 조작
| 기능 | 이클립스 | VSCode | 패턴 |
|------|---------|--------|------|
| 줄 복사 | Alt+Shift+↑/↓ | Shift+Alt+↑/↓ | **동일 키, 다른 순서** |
| 줄 이동 | Alt+Shift+↓ | Alt+↑/↓ | Alt 기반 |
| 줄 삭제 | Ctrl+D | Ctrl+X/Shift+K | Ctrl 기반 |

---

### 4. F-키 기능 패턴

기능 키는 IDE마다 다르지만 유사한 기능을 수행합니다:

| 기능 | 이클립스 | VSCode | 공통적 목적 |
|------|---------|--------|-------------|
| 정의로 이동 | F3 | F12 | 선언부 찾기 |
| 참고 찾기 | - | Shift+F12 | 사용처 찾기 |
| 이름 변경 | - | F2 | 리팩토링 |

---

### 5. 탐색 패턴 (Navigation Patterns)

#### Ctrl+Shift+O의 이중성
- **이클립스**: Import 정리 (Organize Imports)
- **VSCode**: 심볼로 이동 (Go to Symbol)
- **공통점**: 파일/코드 구조 정리 및 탐색

#### 빠른 파일 열기
- **VSCode**: Ctrl+P
- **이클립스**: Ctrl+Shift+R (Resource 열기)
- **공통점**: Ctrl+[수정자]+[문자] 조합

## 학습 전략

### 1. 패턴 기반 학습
개별 단축키를 외우기보다 **패턴**을 이해하세요:
- **Ctrl+Shift+[방향키]**: 선택 확장
- **Alt+[수정자]+[방향키]**: 줄 조작
- **Ctrl+[문자]**: 기본 명령

### 2. 수정자 우선순위
1. **Ctrl**: 가장 기본적인 명령
2. **Shift**: 선택, 확장, 반대 방향
3. **Alt**: 대체, 옵션, 고급 기능

### 3. 토글 vs 단발성
- **토글형**: 같은 단축키 다시 누르면 해제 (Alt+Shift+A)
- **단발형**: 한 번만 실행 (대부분의 단축키)

## IDE별 특징

### 이클립스 특징
- F-키 적극 활용 (F3, F4 등)
- Java 중심 기능 (Import 정리 등)
- 워크박스 기반 단축키

### VSCode 특징
- 복합키 많이 사용 (Ctrl+K, Ctrl+[문자])
- 확장 프로그램과 연동
- 멀티 커서 기능 풍부
- macOS 지원 (Cmd 기반)

## 참고사항

- 단축키는 IDE 버전, 운영체제, 언어 설정에 따라 다를 수 있음
- 충돌하는 단축키는 사용자 정의로 변경 가능
- macOS에서는大部分 Ctrl → Cmd로 대체
- 확장 프로그램 설치 시 충돌 확인 필요
