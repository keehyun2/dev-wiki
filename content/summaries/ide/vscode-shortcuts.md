# VSCode 태그 선택 및 코드 정렬 단축키

## 핵심 단축키

### 태그 쌍 선택
#### 더블 클릭
가장 직관적인 방법으로, 태그 이름을 더블 클릭하면 여는 태그와 닫는 태그가 동시에 선택됩니다.

- **사용법**: `<div>` 같은 태그 이름을 더블 클릭
- **동작**: 여는 태그와 닫는 태그가 모두 선택됨

#### Alt+Shift+→
Emmet 기능과 함께 사용하여 태그 쌍을 선택할 수 있습니다.

### 코드 자동 정렬

#### Shift+Alt+F
VSCode의 가장 중요한 단축키 중 하나입니다.

- **기능**: 현재 파일 전체 자동 정렬 (Format Document)
- **지원**: JavaScript, TypeScript, HTML, CSS, Python 등 다양한 언어
- **연동**: Prettier, ESLint, Black 등 포맷터와 자동 연동

#### Ctrl+K, Ctrl+F
선택된 영역만 정렬하고 싶을 때 사용합니다.

## 필수 단축키

### 선택 및 편집
- **Ctrl+D**: 다음 동일 단어 선택 및 커서 추가
- **Alt+Click**: 여러 위치에 커서 생성
- **Ctrl+/**: 주석 토글
- **Alt+↑/↓**: 현재 줄 이동

### 탐색
- **Ctrl+P**: 파일 빠른 열기
- **Ctrl+G**: 줄 번호로 이동
- **F12**: 정의로 이동

### 터미널 및 패널
- **Ctrl+`**: 터미널 토글
- **Ctrl+B**: 사이드바 토글

## 확장 기능

### 추천 확장
- **Auto Rename Tag**: 여는 태그 수정 시 닫는 태그 자동 수정
- **Prettier**: 코드 자동 정렬
- **ESLint**: JavaScript/TypeScript 린터

### 단축키 설정
- **Ctrl+K, Ctrl+S**: 키보드 단축키 설정 열기
- 필요에 따라 단축키 커스터마이징 가능
