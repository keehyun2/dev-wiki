# 이클립스 태그 쌍 선택 단축키

## 핵심 단축키

### Alt+Shift+A
HTML/XML 에디터에서 여는 태그와 닫는 태그를 동시에 선택하는 단축키입니다.

- **사용법**: 커서를 태그 내부에 위치시키고 `Alt+Shift+A` 누름
- **동작**: 여는 태그와 닫는 태그가 모두 선택됨
- **해제**: 동일한 단축키를 다시 누르면 선택 해제 (Toggle)

### 더블 클릭 방법
태그 이름을 더블 클릭해도 동일한 효과를 얻을 수 있습니다.

## 관련 단축키

- **Ctrl+Shift+F**: 코드 자동 정렬 (Format)
  - 현재 파일 전체 또는 선택된 영역 자동 정렬
  - Java, XML, HTML, JavaScript 등 다양한 파일 타입 지원
  - 코드 스타일 일관성 유지에 필수
- **Ctrl+Shift+→/←**: 단어 단위 선택 확장
- **Ctrl+Shift+Home/End**: 문서 전체 선택
- **Alt+Shift+↑**: 현재 줄 복사
- **Ctrl+Space**: 자동 완성

## 문제 해결

한글 입력 시 문제가 발생하면:
- Window → Preferences → General → Keys
- "Select Enclosing Element" 검색
- 원하는 단축키로 변경
